SkyLoyal: Loyalty Rescue

A working five-agent organisation that predicts which frequent flyers are about to lapse, designs an Octalysis-driven intervention journey, builds the dashboard, writes the marketing pack, and reports it commercially. Researcher to Designer to Maker to Communicator to Manager. Unbroken pipeline.

The pipeline

Five agents collaborate by handing off typed markdown artefacts. Each agent's output is the next agent's sole input. The Manager runs quality gates between every handoff and maintains a cycle log. This site is the third artefact; the others live in the repository.

Octalysis at a glance

The flyer-facing view explicitly tags every gamification element with its Octalysis Core Drive. The flyer view exposes seven of the eight drives (CD6 Scarcity is deliberately omitted from the pilot, no fake countdowns). Every CD-bearing element carries a data-cd="CD<n>" attribute for audit.

  • CD1 Epic Meaning: Sustainable Aviation Fuel attribution.
  • CD2 Development: tier-progress ring, retention progress.
  • CD3 Empowerment: trip-mode picker, choice gestures.
  • CD4 Ownership: miles balance, year-in-review.
  • CD5 Social Influence: peer cohort recognition, alliance pool.
  • CD7 Unpredictability: real surprise upgrades, never fake.
  • CD8 Loss Avoidance: framed as opportunity, not threat.
